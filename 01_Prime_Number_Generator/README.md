# Prime Number Generator

純前端的 JavaScript 演算法應用展示專案。本專案透過直覺的網頁操作與 JavaScript 演算法，即時且動態地生成並過濾出使用者所指定範圍或數量的質數 (Prime Numbers)。

## 技術特點

*   **無框架依賴**：100% 使用 Vanilla JavaScript 操作 DOM，反應迅速，結構單純。
*   **演算法基礎**：實作判斷質數的基礎迴圈，或可擴展為埃拉托斯特尼篩法 (Sieve of Eratosthenes) 等更高階之演算法應用。
*   **網頁呈現**：透過基礎的 HTML/CSS 切版，將計算出的結果動態注入清單。

## 如何執行

這是一個純前端網頁專案。
請直接以任何現代瀏覽器開啟目錄中的 `index.html` 即可使用。
無須安裝任何後端或 Node.js 套件。
